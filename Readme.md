# Coffee Order

AI-powered meeting notes for Mac.

![Coffee Order](website/public/hero.png)

Coffee Order listens to your calls — Zoom, Meet, Teams — and uses AI to generate headings, todos, callouts, and quotes. Ready before you close the tab.

## Development

**Electron app** (root):

```bash
npm install
npm run dev
```

**Website** (`website/`):

```bash
cd website
npm install
npm run dev
```
